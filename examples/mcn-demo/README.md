# Vesper MCN Demo Examples

## Current Demos

## 0. One Sentence to Operation Demo

File:

01-one-sentence-to-operation.md

This demo shows the core idea of Vesper:

A single business question can be mapped into operational objects, reports, follow-up tasks, and approval boundaries.

Example:

> Why is this livestream not converting?

Vesper should not simply answer the question.

It should structure the operation behind the question.
